# PPF Vehículos - Frontend

Aplicación web para el registro de vehículos y control de entradas/salidas, desarrollada con React.js.

## Tecnologías utilizadas

- React.js + Vite
- React Router DOM
- PrimeReact (componentes UI)
- Axios (peticiones HTTP)

## Requisitos previos

- Node.js instalado
- La API (backend) corriendo en `http://localhost:3000`

## Instalación y ejecución

1. Clonar el repositorio
```bash
git clone https://github.com/alexalvarenga7/ppf-vehiculos-frontend.git
cd ppf-vehiculos-frontend
```

2. Instalar dependencias
```bash
npm install
```

3. Correr el proyecto
```bash
npm run dev
```

4. Abrir en el navegador
```
http://localhost:5173
```

## Páginas de la aplicación

- **Vehículos** - Listado, registro, edición y eliminación de vehículos
- **Nuevo Registro** - Registrar entrada o salida de un vehículo
- **Historial** - Ver y filtrar entradas/salidas por fecha, vehículo y motorista

## Nota

Asegurar de tener el backend corriendo antes de usar la aplicación.
